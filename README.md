# dash-survivor
A Vampire Survivors–like web game built with Phaser.  Developed using AI-assisted vibe coding.

## Repository Responsibilities
- `src/`: source code (scenes, entities, systems, config, UI logic)
- `assets/`: runtime source assets used by Phaser loaders
- `assets/vendor/`: third-party/raw source packs and attribution
- `docs/`: design and gameplay specs grouped by domain
- `dist/`: generated build output (do not edit manually)

Specs and design docs are in `docs/`.
