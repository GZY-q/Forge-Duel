# ForgeDuel 命令

## 常用命令

| 命令 | 作用 |
|------|------|
| `npm run dev` | 启动前端开发服务器 (HMR) |
| `npm run server` | 启动后端服务器 |
| `npm run dev:full` | 前端 + 后端同时运行 |
| `npm run build` | 生产构建 |
| `npm run preview` | 预览构建结果 |

## 测试

| 命令 | 作用 |
|------|------|
| `npm run pw:test` | 运行所有测试 |
| `npm run pw:test:smoke` | 冒烟测试（快速） |
| `npm run pw:test:headed` | 打开浏览器测试 |

## 端口

- 前端开发: `localhost:3000`
- 后端服务: `localhost:3001`
- 预览构建: `localhost:4173`
