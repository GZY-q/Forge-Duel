# ForgeDuel - 宇宙战机生存游戏 功能规格书

## 游戏概述

一款类似吸血鬼幸存者的俯视角生存游戏，背景设定在宇宙中。玩家驾驶各种太空战机抵御外星生物入侵。

---

## 核心功能

### 1. 战机系统

- 多种战机选择（至少4种基础战机）
- 每种战机拥有不同的基础属性
  - 生命值
  - 移动速度
  - 初始武器
  - 特殊能力
- 战机升级和进化

### 2. 武器系统

- 自动攻击的远程武器
- 多种武器类型
  - 激光类：直线射击
  - 球体类：圆形范围攻击
  - 导弹类：追踪目标
  - 散弹类：扇形多发
  - 能量类：近战/环绕
- 武器升级和进化
- 武器冷却系统

### 3. 敌人系统

- 多种敌人类型
  - 普通敌人：基本追踪玩家
  - 精英敌人：特殊能力/更高属性
  - Boss敌人：强大但有弱点
- 敌人生成机制
  - 从地图边缘生成
  - 随时间增加难度
  - 波次/阶段系统

### 4. 道具系统

- 经验球：击杀敌人掉落
- 生命恢复：治愈玩家
- 护盾充能：临时无敌
- 武器升级：强化当前武器
- 速度提升：临时移速加成

### 5. 升级系统

- 升级选择：每次升级提供3个选项
- 属性提升
- 新武器获取
- 武器强化

### 6. 关卡/地图系统

- 有限地图空间
- 障碍物和掩体
- 地图边界限制

## 联机功能（未来扩展）

### 1. 房间系统

- 创建房间
- 加入房间（房间号/邀请码）
- 房间设置（人数限制、模式）
- 房间内准备状态
- 房主权限（开始游戏、踢人）

### 2. 玩家同步

- 玩家位置同步
- 玩家动作同步（移动、攻击）
- 玩家状态同步（生命、武器）
- 延迟补偿

### 3. 敌人/道具同步

- 敌人生成同步
- 敌人状态同步
- 道具掉落同步
- 碰撞检测同步

### 4. 社交功能

- 玩家名称显示
- 玩家列表
- 简单的聊天系统
- 好友系统（可选）

### 5. 排行榜系统

- 单人最高分记录
- 多人对战排名
- 天梯等级系统
- 历史战绩

## 技术需求

### 客户端

- Web端优先（HTML5/Canvas）
- 支持桌面和移动端
- 触控虚拟摇杆
- 键盘鼠标支持

### 服务端（未来）

- WebSocket实时通信
- 房间管理服务
- 玩家数据存储
- 排行榜服务

### 性能目标

- 60 FPS稳定运行
- 支持同屏50+敌人
- 低网络延迟（<100ms）
