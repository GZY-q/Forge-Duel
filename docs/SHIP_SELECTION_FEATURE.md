# ForgeDuel 战机选择系统功能规格

## 概述

战机选择界面允许玩家在游戏开始前选择不同的战机，每种战机拥有独特的属性和外观。

## 战机解锁系统

| 战机 | 默认状态 | 解锁条件 |
|------|----------|----------|
| 突击者 | 已解锁 | - |
| 重锤 | 未解锁 | 游戏时间超过60秒 |
| 幻影 | 未解锁 | 游戏时间超过120秒 |
| 闪电 | 未解锁 | 累计击杀100个敌人 |
| 毁灭者 | 未解锁 | 累计击杀500个敌人 |
| 守护者 | 未解锁 | 单局存活超过180秒 |
| 蜂群 | 未解锁 | 达到10级 |
| 终极 | 未解锁 | 累计击杀1000个敌人 |

## 战机选择界面功能

### 显示内容

- 战机缩略图（128x128像素）
- 战机名称
- 难度星级
- 基础属性预览（生命值、速度）
- 解锁状态

### 交互

- 点击战机卡片选择
- 已锁定战机显示解锁条件
- 选中战机高亮显示
- 确认按钮开始游戏

### 状态管理

```
selectedShip: string | null  // 当前选中的战机ID
unlockedShips: string[]         // 已解锁的战机列表
```

## 数据结构

### 战机配置对象

```javascript
{
  id: "striker",
  name: "突击者",
  description: "平衡型战机，速度和火力均衡",
  difficulty: 1,
  initialWeapon: "dagger",
  stats: {
    maxHp: 100,
    speed: 200,
    dashCooldown: 4000
  },
  unlockCondition: null,  // 或 { type: "time", value: 60 }
  iconPath: "assets/sprites/ships/ship_striker_icon.png"
}
```
