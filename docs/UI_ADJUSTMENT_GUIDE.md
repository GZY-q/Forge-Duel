## 📐 快速调整示例

### 调整升级面板大小
```javascript
// GameScene.js 第5628-5629行
const panelW = 400;  // 更宽
const panelH = 450;  // 更高
```

### 调整升级面板选项间距
```javascript
// GameScene.js 第5733行
const optGap = 12;  // 更大的间距
```

### 调整暂停面板按钮大小
```javascript
// GameScene.js 第6011行
宽度: 80 → 100  // 更大按钮
高度: 28 → 36
```

### 调整角色选择面板卡片大小

```javascript
// ShipSelectionScene.js 第79-80行
const cardW = 120;  // 更窄
const cardH = 100;  // 更矮
```
