## 调整提示词的建议

如果生成的图标不符合预期，可以调整以下内容：

1. **颜色**：修改十六进制颜色代码（如 #c4a040）
2. **大小**：改为 128x128 或 32x32
3. **风格**：添加 "more detailed" 或 "simpler" 来调整复杂度
4. **效果**：添加 "with glow effect" 或 "without glow"
5. **背景**：确保是 "transparent background"
