 # ForgeDuel 游戏技能图标生成提示词

## 游戏背景信息
- **游戏名称**: ForgeDuel
- **游戏类型**: 动作Roguelike游戏，类似吸血鬼幸存者（Vampire Survivors）
- **美术风格**: 复古像素风格（Retro Pixel Art Style）
- **字体**: ZpixOne（中文像素字体）
- **主题色调**:
  - 主色: #c4a040 (金色)
  - 强调色: #fef08a (亮黄色)
  - 背景色: #3a3a4a (深灰蓝)
  - 文字色: #ffffff (白色)
  - 边框色: #000000 (黑色)
- **图标尺寸**: 建议 64x64 像素或 128x128 像素
- **输出格式**: PNG 透明背景，图标居中

## 图标设计原则
1. **像素风格**: 使用清晰的像素边缘，不要使用平滑的抗锯齿
2. **简洁明了**: 图标应该快速传达技能含义
3. **高对比度**: 确保在深色背景上清晰可见
4. **金边框**: 图标可以有金色的装饰边框或光晕
5. **单色或少色**: 建议每个图标使用2-3种主要颜色

## 重要提示

1. **保持一致性**: 所有图标应该使用相同的像素风格和调色板
2. **简洁设计**: 像素艺术在缩小时需要保持清晰，细节不要太多
3. **测试缩放**: 在游戏界面中测试图标在不同尺寸下的显示效果
4. **反馈调整**: 根据实际效果调整提示词，直到获得满意的图标
